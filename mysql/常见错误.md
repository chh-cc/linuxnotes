# 常见错误

1. 数据库远程连接很慢

   原因：mysql对连接的客户端进行DNS反向解析

   解决：

   ```shell
   [mysqld]
   skip-name-resolve
   增加参数后msyql授权表中的host字段就不能使用域名只能用ip
   ```

2. 远程连接不上

   原因：默认的bind-address是127.0.0.1

   解决：在bind-address后面增加远程访问的ip

3. 数据库乱码

   查看配置是否字符集统一，不统一根据自行调整

   ```mysql
   show variables like 'character_set_%';
   ```
