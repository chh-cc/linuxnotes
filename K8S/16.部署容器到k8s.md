## 部署单实例redis

访问dockerhub找官方镜像

![image-20220123141208511](https://gitee.com/c_honghui/picture/raw/master/img/20220123141215.png)

创建配置文件的configmap

```shell
kubectl create cm redis-conf --from-file=/root/k8s-file/redis.conf --namespace=text
```

创建deployment

```yaml
cat redis-single-node.yaml

apiVersion: apps/v1
kind: Deployment
metadata:
  labels:
    app: redis-single-node
  name: redis-single-node
  namespace: text
spec:
  progressDeadlineSeconds: 30
  replicas: 1
  revisionHistoryLimit: 10
  selector:
    matchLabels:
      app: redis-single-node
  strategy:
    rollingUpdate:
      maxSurge: 1
      maxUnavailable: 0
    type: RollingUpdate
  template:
    metadata:
      creationTimestamp: null
      labels:
        app: redis-single-node
    spec:
      containers:
      - image: redis:5.0.4-alpine
        imagePullPolicy: IfNotPresent
        name: redis-single-node
        ports:
        - containerPort: 6379
          name: redis
          protocol: TCP
        command:
        - sh
        - -c
        - redis-server "/mnt/redis.conf"
        env:
        - name: TZ
          value: Asia/Shanghai
        - name: LANG
          value: C.UTF-8
        resources:
          requests:
            cpu: 10m
            memory: 50Mi
          limits:
            cpu: 1000m
            memory: 1024Mi
        readinessProbe:
          tcpSocket:
            port: 6379
          initialDelaySeconds: 10
          timeoutSeconds: 2
          periodSeconds: 10
          successThreshold: 1
          failureThreshold: 2
        livenessProbe:
          tcpSocket:
            port: 6379
          initialDelaySeconds: 10
          timeoutSeconds: 2
          periodSeconds: 10
          successThreshold: 1
          failureThreshold: 2
        volumeMounts:
        - name: redis-conf
          mountPath: /mnt/
          readOnly: True
      dnsPolicy: ClusterFirst
      restartPolicy: Always
      schedulerName: default-scheduler  
      volumes:
      - name: redis-conf
        configMap:
          name: redis-conf
```

创建一个service，命名格式：中间件名-项目名，不要只用中间件名

```yaml
apiVersion: v1
kind: Service
metadata:
  name: redis-single-node
  namespace: text
spec:
  ports:
  - port: 6379
    protocol: TCP
    targetPort: 6379
  type: ClusterIP
  selector:
    app: redis-single-node
```
